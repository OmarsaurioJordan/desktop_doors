# Admin Doors

Requiere Godot 4

Este software está hecho con el propósito de ser un modelo apróximado, de módulo administrador de credenciales de acceso, para puertas automátizadas, se ha creado para el semillero investigativo del CEAI del SENA en 2025, desde el programa ADSO. 

En principio la tarea era crear un mockup interactivo en figma, pero con el propósito de hacer pruebas de usabilidad, se optó por un desarrollo software ágil, por lo mismo varias funcionalidades se han hecho con minimalismo.

La interfaz no se hizo responsiva, pero es algo que si es posible en Godot.

Se adjunta un documento de investigación y diseño, así como otro texto markdown con las tareas pendientes. Este software al estar basado en MVC, tiene capacidades de ser conectado a una base de datos externa, ejemplo, en el servidor XAMPP, para hacer prubas mejoradas. Así mismo puede conectarse a periféricos como Arduino para emular el funcionamiento de las puertas reales.
